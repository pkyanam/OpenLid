# Development

## Requirements

- macOS 14 or newer.
- Xcode 16 or newer. The project file was last upgraded with Xcode 26.3.
- Swift package resolution access for `https://github.com/soffes/HotKey`.

## Build

Open the project in Xcode:

```sh
open OpenLid.xcodeproj
```

Build from the command line:

```sh
xcodebuild -project OpenLid.xcodeproj -scheme OpenLid -configuration Debug build
```

There are currently no test targets in the project.

## Common Development Tasks

Add a process-detected default agent:

1. Edit `AppConfig.defaultAgents`.
2. Add an `Agent` with `detection: AgentDetection(processNames: [...])`.
3. Use exact process/app names (matching is exact, case-insensitive).
4. Document it in `docs/configuration.md`.

Add a hook-detected default agent:

1. Edit `AppConfig.defaultAgents`.
2. Add an `Agent` with
   `detection: AgentDetection(hookID: "...", processNames: ["..."])` (process names
   are the fallback before the hook is installed).
3. Add a hook script and README under `examples/hooks/`.
4. Update `examples/hooks/README.md` and `docs/agent-hooks.md`.

Change a setting:

1. Add or update a field in `AppConfig`.
2. Expose it in `SettingsView`.
3. Use it from `AppState` or the relevant service.
4. Keep default behavior safe when old JSON config lacks the field.

## Manual Verification Checklist

- Build succeeds in Debug.
- App appears only in the menu bar, not the Dock.
- `Option+Command+L` toggles the header state.
- A manual hook file causes the matching agent to show as working.
- Removing the hook file starts the idle timeout and releases the assertion.
- Battery guardrails block holds when expected.
- Auto-sleep prompts for Automation permission if needed and then requests sleep.
